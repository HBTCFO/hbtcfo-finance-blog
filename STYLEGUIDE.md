# HBTCFO Finance — Style Guide (живые правила)

Этот файл — "память" редактора. Его можно пополнять в режиме обучения.

## Как добавлять правила (чтобы бот понял)
Пиши в личку @FinanceSecondBrainBot или в учебный чат сообщения, которые начинаются с одного из префиксов:

- ПРАВИЛО: <одно правило>
- ЗАПРЕТ: <что нельзя>
- ШАБЛОН: <шаблон структуры>
- ПРИМЕР_ХОРОШО: <пример текста>
- ПРИМЕР_ПЛОХО: <пример текста>
- ТОН: <как звучим>
- ФАКТЫ_РФ: <как писать про РФ (ОФЗ/ЦБ/рубль/эмитенты)>

Бот обязан:
- сохранить это в этот файл;
- подтвердить, что правило добавлено;
- при генерации постов для сайта/группы всегда сначала читать этот файл и строго соблюдать.

## Базовые принципы (по умолчанию)
- Пишем по-русски естественно, без англицизмов там, где можно.
- Структура > поток сознания. Буллеты приветствуются.
- Без "служебных хвостов": никаких "если хочешь - могу...", вопросов к читателю, призывов обсудить.
- Источники: 1-2 ссылки максимум, только по факту.
- Длинное тире "—" не использовать (только обычный дефис "-").
